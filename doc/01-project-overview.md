# 프로젝트 개요

## 프로젝트 정보

**프로젝트명:** ScaledTales (스케일드테일즈)  
**설명:** 파충류/양서류 사육 관리 앱  
**버전:** 1.0.0 (MVP)  
**개발 기간:** 8주 (약 2개월)  
**개발 방식:** 1인 개발 (Claude Code 활용)

## 프로젝트 목표

이색 애완동물, 특히 **파충류와 양서류**를 사육하는 사람들을 위한 종합 사육 관리 애플리케이션을 개발합니다.

### 핵심 가치
- **체계적인 사육 기록**: 개체별 상세한 사육 일지 작성 및 관리
- **건강 모니터링**: 무게/길이 측정을 통한 성장 추적
- **브리딩 관리**: 짝짓기부터 부화까지 전 과정 기록
- **합법적 사육 지원**: 백색목록 및 CITES 등재 종 정보 제공

## 타겟 사용자

### 1차 타겟 (MVP)
- **초보 사육자**: 처음 파충류/양서류를 키우는 사람들
- **중급 사육자**: 여러 개체를 체계적으로 관리하고 싶은 사람들

### 2차 타겟 (추후 확장)
- **전문 브리더**: 번식 기록 및 계통 관리가 필요한 사람들
- **판매자**: 개체 판매 이력 관리

## 핵심 기능

### 1. 회원 관리
- 회원 가입 및 로그인 (Supabase Auth)
- 프로필 관리
- 다국어 지원 (한국어/영어)

### 2. 개체 관리
- **개체 등록**: 종, 모프, 성별, 생년월일, 출신 정보 등
- **개체 목록**: 필터링, 정렬, 검색
- **개체 상세**: 기본 정보, 측정 기록, 사육 일지 통합 조회
- **개체 수정/삭제**: 정보 업데이트 및 상태 관리 (생존/사망/분양)

### 3. 사육 일지 관리
7가지 이벤트 타입별 상세 기록:

#### 기본 일지
- **먹이 급여**: 먹이 종류, 수량, 보충제, 급여 반응
- **탈피**: 완성도, 문제 부위, 도움 방법, 습도
- **배변**: 대변 상태, 요산 상태, 이상 소견

#### 브리딩 일지
- **짝짓기**: 파트너 정보, 성공 여부, 행동 관찰
- **산란**: 알 개수, 부화 계획, 온습도 설정
- **검란**: 유정란/무정란/중지란 개수 기록
- **부화**: 성공/실패 개수, 부화 개체 연결

### 4. 측정 기록
- 무게/길이 측정 이력 관리
- 성장 그래프 시각화
- 최신 측정값 자동 업데이트

### 5. 종 분류 시스템
- 계층적 분류 (강 → 목 → 과 → 속 → 종)
- 학명 및 일반명 제공
- 백색목록 및 CITES 등재 정보
- 검색 및 필터링

## MVP 범위

### 포함 기능 ✅
- 회원 가입/로그인
- 개체 CRUD (등록/조회/수정/삭제)

## 기술적 특징

### 현대적 기술 스택
- **백엔드**: TypeScript + NestJS (체계적인 아키텍처)
- **프론트엔드**: Flutter (크로스 플랫폼)
- **데이터베이스**: PostgreSQL (Supabase)
- **인증**: Supabase Auth (OAuth, JWT)
- **스토리지**: Supabase Storage (이미지)

### 설계 원칙
- **RESTful API**: 표준화된 엔드포인트 설계
- **정규화된 DB**: 확장 가능한 스키마 구조
- **타입 안정성**: TypeScript, Dart 활용
- **컴포넌트 재사용**: 모듈화된 코드 구조
